# MineSharp

## What does it do?

MineSharp is a simple and quite bad looking Minecraft Launcher, written in C#.\
But there's a twist: everything happens in the console!

Everything works exacty like you would expect a GUI to work, but it's just... in the *Console*!

## Setup Guide

1. Download the installer `MineSharp-Install.msi` from the releases page
2. Execute the file you just downloaded and follow the instructions given
3. Start the MinSharp Launcher
4. Press `Sign In`, if you don't want to play cracked
5. The rest is probably similar to the Modrinth App... Just way worse!
